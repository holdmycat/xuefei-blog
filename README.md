# xuefei-blog
Personal tech blog &amp; game dev notes (Unity, AOT, Skill Systems, AI tools).

## Local development

### Prerequisites
- Hugo (extended) installed locally (`hugo version`)

### Start the dev server
From repo root:
- `cd blog && hugo server -D -F`

Then open:
- `http://localhost:1313/xuefei-blog/`

### Quick start script
For convenience, run from repo root:
- `bash ./start-local.sh`

You can pass extra Hugo args, e.g.:
- `bash ./start-local.sh --port 1314`
