Modular Skill System (Core)

Designer-first, data-driven combat skill framework for Unity (IL2CPP-friendly).

What it is

Modular runtime + data-first authoring for combat skills, with guardrails for determinism and AOT safety.

Why it matters

Speeds iteration, keeps data consistent across designers, and ships safely on IL2CPP platforms.

Key highlights

  • Composable action modules (spawn/FX/physics/state) driven by ScriptableObject data.
  • Editor validators catch reflection/AOT risks and missing assets before build.
  • Deterministic math + prewarm for IL2CPP; netcode/rollback-friendly.
  • Authoring timeline to preview timings/payloads without playmode.
  • Deploy guardrails: lint, bake, and report for skills per build.
