Modular Skill System

data-driven combat modules for rapid iteration

Problem

Designers and engineers needed a reusable way to build, ship, and tune combat skills without rewriting code for every variant.

Solution

Built a modular skill runtime with data-first authoring, guardrails for determinism/AOT, and editor tooling for validation.

Key Design Decisions

  • Action modules compose spawn/FX/physics steps driven by ScriptableObject data.
  • Editor validators catch reflection/AOT risks and missing assets before build.
  • Runtime prewarm for IL2CPP and deterministic math to support netcode/rollback.
  • Authoring timeline to preview timings and payloads without entering playmode.
